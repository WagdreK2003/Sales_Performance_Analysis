# Sales Performance Analysis Dashboard

This project analyzes retail sales data to identify revenue trends, regional performance, and product-level profitability.

## Tools Used
- Excel (Data Cleaning)
- SQL (Data Analysis)
- Power BI (Data Visualization)

## Key Features
- KPI metrics: Total Sales, Total Profit, Orders, Profit Margin
- Monthly sales trend analysis
- Region-wise and category-wise performance
- Interactive slicers for year, region, and category
